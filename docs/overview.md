# Overview

This project collects the source code, documentation and tests for the Paper Trading BOTo framework.  The original long form documentation was located under `paper_trading_boto/README.md`, which can be cumbersome to browse when using the GitHub interface.  To make navigation easier, the documentation has been consolidated here in the `docs` directory.

For the complete feature list, installation guide, and usage examples, please see the original [README](../paper_trading_boto/README.md).
